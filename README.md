# ZOOMIES

## Summary

ZOOMIES is a colorful endless runner built for Apple Watch. Choose a character and map, jump over obstacles, collect coins, and chase your best distance.

## Features

- Multiple playable characters and themed maps
- Standard and one-life YOLO modes
- Coin collection, persistent totals, and best-distance tracking
- Sound, haptic, control, and distance-unit settings
- Animated watchOS interface
